# US-110 - Shipping Address Saving (Preliminary)

---

## Identificación

- **ID:** US-110
- **Fecha:** 2025-03-11
- **Servicio:** cart-service

---

## User Story

Como usuario recurrente quiero poder guardar mi dirección de envío para no tener que introducirla en cada compra.

---

## Descripción

Se propone habilitar la funcionalidad de guardado de dirección de envío para mejorar la experiencia de usuario durante el proceso de compra. Actualmente, la dirección solo se utiliza durante la sesión y no se almacena para futuros pedidos. Existen varias alternativas a considerar, como guardar la última dirección usada, permitir guardar varias direcciones y limitar la funcionalidad a usuarios logueados. No se ha definido aún cuál alternativa se implementará ni los detalles técnicos o legales asociados. Esta User Story es preliminar y está sujeta a definición por parte de Producto.

---

## Cambios

### Qué se añadió

- Propuesta preliminar para permitir el guardado de dirección de envío para futuras compras.
- Consideración de diferentes alternativas de implementación (última dirección, múltiples direcciones, solo usuarios logueados).
