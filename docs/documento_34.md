# Documento 34

## Tipo de Documento
Procedimiento Interno

## Área
Atención al Cliente – E-commerce

## Código
PROC-DEV-014

## Descripción
Procedimiento para la gestión de devoluciones cuando el cliente no conserva el embalaje original del producto.

## Pasos
1. Verificar el número de pedido en el sistema.
2. Confirmar que el producto no pertenece a la categoría de exclusiones.
3. Evaluar el estado físico del artículo devuelto.
4. Registrar incidencia como “Devolución sin embalaje”.
5. Escalar a supervisión si el valor supera los 150€.

## Observaciones
Este procedimiento aplica únicamente a compras realizadas a través del canal online.

## Vigencia
Documento vigente desde 2023-11-01
