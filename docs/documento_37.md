# Documento 37

## Tipo de Documento
Procedimiento Interno

## Área
Fraude y Seguridad – Comercio Digital

## Código
PROC-FRD-008

## Descripción
Pasos a seguir ante la detección de pedidos con patrones de fraude potencial.

## Criterios de Detección
- Múltiples intentos de pago fallidos.
- Dirección de envío distinta a la dirección de facturación.
- Uso de métodos de pago internacionales no habituales.

## Pasos
1. Marcar el pedido como “En revisión”.
2. Bloquear el envío automáticamente.
3. Notificar al cliente por correo estándar.
4. Revisar manualmente en un plazo máximo de 24h.

## Observaciones
No se debe cancelar el pedido sin validación previa del equipo de fraude.
