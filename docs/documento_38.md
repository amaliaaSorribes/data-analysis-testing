# Documento 38

## Tipo de Documento
FAQ Interna

## Área
E-commerce

## Pregunta
¿Se puede cambiar el método de pago una vez confirmada la compra?

## Respuesta
No es posible modificar el método de pago tras la confirmación. La única opción es cancelar el pedido (si aún no ha sido enviado) y realizar uno nuevo.

## Observaciones
Esta información debe comunicarse claramente al cliente.
