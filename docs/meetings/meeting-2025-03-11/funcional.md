# Documento funcional – Guardado de dirección de envío (2025-03-11)

## Descripción General

Se plantea la posibilidad de guardar la dirección de envío para futuras compras, ya que algunos usuarios han solicitado no tener que introducir la dirección en cada compra. Se reconoce que esta funcionalidad podría mejorar la experiencia de usuario, aunque no se ha determinado su prioridad ni su impacto en conversión.

## Lógica Funcional

Se han propuesto varias alternativas para el guardado de direcciones:
- Guardar la última dirección usada.
- Permitir guardar varias direcciones.
- Limitar la funcionalidad solo a usuarios logueados.

No se ha tomado una decisión sobre cuál de estas alternativas implementar.

---

## Lógica Backend

Actualmente, el backend no guarda la dirección de envío; solo se utiliza durante la sesión. No se ha definido aún cómo se debería implementar el guardado de direcciones.
