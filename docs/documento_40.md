# Documento 40

## Tipo de Documento
Procedimiento Interno

## Área
Logística y Postventa

## Código
PROC-LOG-021

## Descripción
Procedimiento para aplicar reembolsos parciales cuando el pedido llega incompleto o con retraso.

## Casos Aplicables
- Falta de uno o más artículos.
- Retraso superior a 72 horas sobre la fecha prometida.

## Pasos
1. Confirmar incidencia en el sistema logístico.
2. Validar reclamación del cliente.
3. Calcular importe a reembolsar.
4. Ejecutar reembolso por el mismo método de pago.

## Observaciones
El reembolso parcial no implica cancelación del pedido completo.
